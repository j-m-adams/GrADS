set background

set background color#

Sets the background color to color#.

Usage Notes

  1. This command is used instead of set display when a color other than black or white is desired for the background color.
  2. The new background is not drawn until a clear command is issued.
  3. If you use set background, then the options 'black' and 'white' for the gxprint command cannot be used to override the background color.
